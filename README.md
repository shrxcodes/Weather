# 🌤️ Weather App
A simple and clean Weather Application built using **HTML**, **CSS**, and **JavaScript**.  
It uses the **OpenWeatherMap API** to fetch real-time weather details of any city in the world.
## 🚀 Features
- Search weather by city name  
- Displays:
  - 🌡️ Temperature (°C)
  - 💧 Humidity
  - 🌬️ Wind Speed
  - 🌤️ Weather Condition Icon
- Error handling for invalid city names
- Clean and responsive UI
## 🛠️ Technologies Used
- **HTML5**  
- **CSS3**  
- **JavaScript (Fetch API)**  
- **OpenWeatherMap API**
  
## 📦 How It Works
1. User enters a city name  
2. App sends an API request to OpenWeatherMap  
3. Receives JSON data  
4. Displays:
   - City name  
   - Temperature  
   - Humidity  
   - Wind speed  
   - Weather condition  
5. Shows icons based on weather type (Clear, Clouds, Rain, etc.)
